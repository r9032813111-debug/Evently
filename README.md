# Evently
Evently — Платформа для локальных событий

**Место, где люди собираются вместе.**

## ✨ Возможности

- **🔐 Авторизация и регистрация** пользователей
- **📅 Создание событий** с фотографиями и подробным описанием
- **📍 Удобный выбор даты и времени** (календарь)
- **👥 Регистрация участников** на события
- **👤 Личный кабинет** с созданными событиями и регистрациями
- **🗑 Удаление своих событий** автором
- **📱 Адаптивный современный дизайн** на Bootstrap 5

---

## 🛠 Технологии

- **Backend**: Python + Flask
- **База данных**: SQLAlchemy + SQLite
- **Авторизация**: Flask-Login
- **Формы**: Flask-WTF + WTForms
- **Frontend**: Bootstrap 5 + Jinja2
- **Загрузка файлов**: Werkzeug
- **Форматирование**: Black

---

## 🚀 Быстрый запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ваш_ник/evently.git
cd evently

# 2. Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите проект
python run.py
```

Перейдите в браузере: **http://127.0.0.1:5000**
